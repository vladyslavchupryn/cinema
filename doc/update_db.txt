1. Настраиваем обновление БД
вот возможные плагины: http://stackoverflow.com/questions/2809422/how-to-database-versioning-with-maven2
1) для обновления нужно писать xml http://www.liquibase.org/manual/maven (+ на сайте ошибка с размещение блока configuration в инструкции)
2) нет возможности использовать без maven https://code.google.com/p/c5-db-migration/


был выбран этот: http://flywaydb.org/documentation/migration/sql.html

есть статья: http://habrahabr.ru/post/141354/

a) настроить pom
в данном случае файлы миграций должны лежать в src/main/db/migration (именование описано тут http://flywaydb.org/documentation/migration/sql.html)



               <plugin>
                    <groupId>com.googlecode.flyway</groupId>
                    <artifactId>flyway-maven-plugin</artifactId>
                    <version>2.1.1</version>
                    <configuration>
                         <url>${jdbc-url}</url>
                         <user>${jdbc-username}</user>
                         <password>${jdbc-password}</password>
                         <table>monitoring</table>
                         <locations>
                              <location>filesystem:src/main</location>
                         </locations>
                         <!--<locations>resources</locations>-->
                    </configuration>
               </plugin>

<profile>
   <id>main</id>
   <properties>
<!-- просто jdbc:postgresql:monitoring не работает при создании БД -->
                  <jdbc-url>jdbc:postgresql://localhost/monitoring</jdbc-url>
<jdbc-username>****</jdbc-username>
<jdbc-password> ****</jdbc-password>
   </properties>
</profile>


b) выполнить обновлени

mvn flyway:migrate

очистить БД

mvn flyway:clean
